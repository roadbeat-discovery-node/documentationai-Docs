This guide walks you through setting up a Discovery Node on your local machine for development and testing.

## Prerequisites

- **Node.js** >= 20 LTS
- **pnpm** >= 9
- **Docker Desktop** (for Elasticsearch, Redis, NATS)
- **PostgreSQL** 18.x (local or Docker)

<Callout kind="tip">
  On Windows, install PostgreSQL natively. On macOS/Linux, Docker works well for all services.
</Callout>

## Setup

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/roadbeat/discovery-node.git
    cd discovery-node
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    pnpm install
    ```
  </Step>

  <Step title="Start infrastructure services">
    Start Elasticsearch 8.11.4, Redis 7, and NATS 2.10 using Docker Compose:
    ```bash
    pnpm docker:dev
    ```
    This starts the services defined in `docker/docker-compose.yml`.
  </Step>

  <Step title="Configure environment">
    Copy the example environment file and adjust as needed:
    ```bash
    cp apps/api/.env.example apps/api/.env
    ```

    Key variables to set:
    ```bash
    DATABASE_URL="postgresql://postgres:password@localhost:5432/roadbeat_discovery_node"
    ELASTICSEARCH_URL="http://localhost:9200"
    REDIS_URL="redis://localhost:6379"
    ADMIN_API_KEY="your-secret-admin-key"
    ```
  </Step>

  <Step title="Run database migrations">
    ```bash
    pnpm db:migrate
    pnpm db:generate
    ```
  </Step>

  <Step title="Start the API server">
    ```bash
    pnpm dev:api
    ```
    The API starts on **port 3300**. Swagger docs are available at [http://localhost:3300/api/docs](http://localhost:3300/api/docs).
  </Step>

  <Step title="Start the website (optional)">
    In a separate terminal:
    ```bash
    pnpm dev:web
    ```
    The transparency website starts on **port 3301**.
  </Step>
</Steps>

## Verify it works

Once the API is running, check the health endpoint:

```bash
curl http://localhost:3300/health
```

You should see a response like:

```json
{
  "status": "healthy",
  "timestamp": "2026-03-10T10:00:00.000Z",
  "components": {
    "postgresql": { "status": "up" },
    "elasticsearch": { "status": "up", "clusterStatus": "green" },
    "redis": { "status": "up" },
    "nats": { "status": "up" }
  }
}
```

## Ports reference

| Service | Port | URL |
|---------|------|-----|
| **Discovery Node API** | 3300 | `http://localhost:3300` |
| **Swagger Docs** | 3300 | `http://localhost:3300/api/docs` |
| **Website** | 3301 | `http://localhost:3301` |
| **Elasticsearch** | 9200 | `http://localhost:9200` |
| **Redis** | 6379 | — |
| **NATS** | 4222 | — |
| **NATS Monitoring** | 8222 | `http://localhost:8222` |
